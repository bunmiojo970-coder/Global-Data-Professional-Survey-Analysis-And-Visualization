# Global-Data-Professional-Survey-Analysis-And-Visualization

# Executive Summary:

Using Microsoft Power BI, I transformed and visualized the global data professional survey dataset creating an interactive dashboard using charts and filters to identify trends in salary expectations and career experiences, delivering insights in order to highlight key areas for industry improvement.
